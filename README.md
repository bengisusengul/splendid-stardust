# ✨ Splendid Stardust · 3 Haftalık Kamp

Kişisel beslenme + antrenman + supplement takip uygulaması.

## Özellikler

- 📅 Bugünün checklist'i (Pzt-Pzr × Hf 1-3)
- 💊 Saat saat supplement çizelgesi
- 🔄 Yedek malzeme tablosu (15+ alternatif)
- 🚨 Tatlı krizi 5 adım protokolü
- 🍽️ Filtre listeleri (Brassica/Solanaceae/tohum yasakları)
- 📊 Haftalık ölçüm takibi + enerji notları
- 💾 LocalStorage ile durum kaydedilir
- 📱 Mobil uyumlu (PWA tarzı ana ekrana eklenebilir)

## Kullanım

Tek HTML dosyası, hiçbir kurulum gerekmez:

1. **Telefonda:** GitHub Pages URL'sini Safari'de aç → Paylaş → "Ana Ekrana Ekle"
2. **Bilgisayarda:** `index.html`'yi tarayıcıda aç

## Veri

Tüm tikledim/ölçüm/enerji notları tarayıcının localStorage'ında saklanır. Tarayıcıyı temizlersen veya başka cihaza geçersen kaybolur.

## Stack

Vanilla HTML + CSS + JavaScript. Hiçbir bağımlılık yok.
